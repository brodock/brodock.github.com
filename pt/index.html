CIMM

Desenvolvedor Full-Stack (Freelancer)
Janeiro 2011 - Janeiro 2012

Reescrita completa da aplicação existente, já que era composta por base de código legada, de dificil manuntenção, originalmente escrita para Ruby on Rails 1.1. Anteriormente a este projeto, esta base de código recebeu correções pontuais que possibilitou sua atualização até a versão 2.1, mas ainda assim, permaneceu com uma enorme quantidade de dívidas técnicas.

O portal foi dividido em aplicações menores, com um escopo bem definido, com a finalidade de facilitar futuras necessidades de escalabilidade e simplificar a base de código existente. As aplicações foram interligadas por uma solução centralizada de autenticação (baseado em CAS).

A base de dados foi migrada do PostgreSQL, para uma combinação de MySQL, destinado aos dados relacionais e MongoDB, destinado aos dados analíticos. Essa alteração resultou em um ganho de performance em ordens de magnitude, principalmente para a geração de relatórios baseados nos dados analíticos (reduziu tempo de geração de aprox. 30 horas para menos de 10 minutos).
